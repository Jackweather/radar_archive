What This Trains

This model is a lightweight logistic regression artifact saved as a single JSON file. It does not need scikit-learn at runtime and can be read later by Python code directly.

Current limitation: this first version learns from radar reflectivity patterns alone. It is useful as a storm-severity scorer, but it is not yet a true warning model because it has no storm-report or warning labels.
  • Read every available GRIB file under the archive path.
  • Sample strong and weak reflectivity neighborhoods from each file.
  • Fit one portable model file: radar_severe_model.json.
